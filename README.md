# mi-primer-repo

Repositorio de práctica para aprender el flujo de trabajo con Git y GitHub.

## 📋 Qué es esto

Este repositorio sirve como espacio de aprendizaje para practicar el ciclo
completo de trabajo con control de versiones:

- Crear ramas para trabajar de forma aislada
- Hacer commits con cambios pequeños y claros
- Abrir Pull Requests (PR) para revisar los cambios antes de integrarlos
- Fusionar (merge) los cambios a la rama principal

## 🌳 Flujo de trabajo

1. **Crear una rama** a partir de `main` para cada cambio.
2. **Hacer commits** describiendo qué se modificó y por qué.
3. **Abrir un Pull Request** hacia `main`.
4. **Revisar y fusionar** una vez que el cambio esté listo.
